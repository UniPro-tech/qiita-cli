## Features

<CardGrid stagger>
  <Card title="好きなエディターで書ける" icon="pencil">
    Qiita CLI は Markdown 形式で記事を書くため、好きなエディターで執筆できます。
  </Card>
  <Card title="見やすいプレビュー" icon="document">
    `qiita preview` コマンドで、ブラウザ上で見やすいプレビューを確認できます。
  </Card>
  <Card title="便利なディレクトリ管理" icon="seti:folder">
    公式のCLIにはない、ディレクトリに記事をまとめて管理する機能があります。
  </Card>
	<Card title="GitHub連携" icon="github">
		GitHubと連携して、リポジトリ内の記事をQiitaに投稿・更新できます。
	</Card>
</CardGrid>
