Qiita CLI は公式の Qiita CLI リポジトリをベースにしていますが、いくつかの独自の変更点があります。以下に主な変更点を示します。

## 🚀 追加機能

ここに、Qiita CLI に追加された主な機能を示します。

### 📁 ディレクトリ分け機能

公式リポジトリではできなかった、**`public`ディレクトリ配下の記事をディレクトリ分けすることができます**。

「毎年あるアドベントカレンダーで記事が多くなってしまった...」なんて場合でも、ディレクトリ分け機能を使えば、記事を整理して管理しやすくなります！

#### 例

ディレクトリ階層:

```
.
└─ public
   ├── newArticle001.md
   ├── newArticle002.md
   └── adventCalendar
       ├── day1.md
       └── 2025
           └── day1.md

```
